The Code for Progress "ideal" applicant

We know that applicants bring different strengths to our program, so we don’t expect everyone to embody all of these characteristics. But there are a few things that we think make people a good fit for our program. The ideal Code for Progress applicant:

  • Wants to use this program as an entry point into the professional social justice movement
  • Has encountered barriers to accessing education, jobs, and resources
  • Likes to solve problems with creative solutions
  • Is good at reading a situation and identifying specific human needs
  • Has patience when working through obstacles
  • Looks for ways to teach and mentor others
  • Is willing to ask for help
  • Likes to figure out how things work
  • Has an idea for a digital tool that can help to address a need in their community or organization
  • Is comfortable using a word processor, email, and web browser
